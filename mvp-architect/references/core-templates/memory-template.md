# MEMORY.md Template

Generate two artifacts during Phase C:

1. `docs/MEMORY.md` - index of dated memory entries.
2. `docs/memory/YYYY-MM-DD.md` - today's handoff entry for the generated planning bundle.

Memory is datable execution state, not decisions and not known issues. Keep decisions in `DECISIONS.md`; keep known bugs/security gaps/tech debt in `KNOWN_ISSUES.md`.

## `docs/MEMORY.md`

```markdown
# MEMORY

Index of `docs/memory/` daily entries. Newest first.

- [YYYY-MM-DD](memory/YYYY-MM-DD.md) - Phase C planning bundle generated by mvp-architect; ready for Phase 1 implementation handoff.
```

